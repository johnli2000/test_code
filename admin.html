快速启用方式

  1. 在服务器上设置 ADMIN_KEY 环境变量。
  2. 启动 npm install && npm start。
  3. 访问 /admin 并填写资料。

所有提交会写入本地 data/models.json，仅服务器可读。

安全建议

  • 限制后台访问 IP（使用反向代理或防火墙）。
  • 确保后台口令足够复杂并定期更换。
  • 通过 HTTPS 访问后台地址。
